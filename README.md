# Portfolio — Siti Nabila

Website portfolio pribadi berbasis HTML/CSS/JS murni, tanpa framework.

## Struktur Folder

```
portfolio-nabila/
├── index.html          ← File utama (buka ini di browser)
├── css/
│   └── style.css       ← Semua styling
├── js/
│   └── main.js         ← GitHub API + animasi
├── assets/
│   └── img/
│       └── foto.jpg    ← ← TARUH FOTO KAMU DI SINI
└── README.md
```

## Cara Pakai

1. **Buka `index.html`** di browser — langsung jalan.
2. **Tambah foto:** taruh file foto kamu di `assets/img/foto.jpg`  
   *(nama file harus sama persis: `foto.jpg`)*
3. **Proyek otomatis** diambil dari GitHub `nabilaasn` via API — tidak perlu ubah apapun.
4. **Deploy gratis:** upload semua file ke [GitHub Pages](https://pages.github.com) atau [Netlify](https://netlify.com).

## Kustomisasi

Edit `index.html` untuk mengubah:
- Teks deskripsi
- Data pengalaman / organisasi
- Link kontak (email, LinkedIn, dll)

Edit `css/style.css` untuk mengubah:
- Warna tema (lihat `:root` di baris atas)
- Font
- Layout

## GitHub Pages (Cara Deploy)

1. Buat repo baru di GitHub (misal: `portfolio`)
2. Upload semua file
3. Settings → Pages → Source: `main` branch → `/root`
4. Website live di `https://nabilaasn.github.io/portfolio`
